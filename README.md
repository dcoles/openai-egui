# OpenAI GUI client (egui)

A small project to learn how to create GUIs using [egui](https://www.egui.rs/).

Not intended for any real-world use.

# Building

```bash
cargo build --release

echo 'OPENAI_API_KEY' > openai.token
target/release/openai-egui
```
